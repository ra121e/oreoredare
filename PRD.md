### **製品要件定義書（PRD） - 「オレオレ誰？」**

#### **1. アプリ概要**

-   **アプリ名**：オレオレ誰？（OreOre Dare?）
-   **ターゲットユーザー**：
    -   メイン：70歳以上の高齢者（スマホ初心者〜中級者）
    -   サブ：家族（子ども・孫） → 高齢者のスマホを遠隔で登録・監視可能
-   **解決する課題**：
    -   オレオレ詐欺・還付金詐欺・声クローン詐欺の被害（2025年も急増中）
    -   高齢者が「家族の声」を信じてしまう心理
    -   知らない番号からの電話/SMSに即対応できない
-   **バリュープロポジション**： 「AIお孫さんが24時間ガードマンになって、怪しい電話を全部撃退。本当に大事な電話だけおばあちゃん/おじいちゃんにパスします。」

#### **2. コア機能（MVP優先順位）**

**Must（MVP必須）**

1.  **AI自動応答ガード**（電話着信）
    -   着信時にAIが即応答（高齢者風の優しい声）
    -   スクリプト例：「もしもし、〇〇（名前）さんのお孫ですけど、どちら様でしょうか？」
    -   相手の返答をリアルタイムで解析 → 詐欺パターン検知（「お父さん？」「事故に遭った」「今すぐ振り込んで」など）
    -   安全判定基準：
        -   事前登録家族の声紋マッチ or 合言葉一致 → 本人に即パス（プッシュ通知 + 自動接続）
        -   怪しい場合 → 「確認しますので折り返しますね」と切り、かけ直し提案
2.  **SMS/メッセージ自動解析**
    -   LINE・SMS受信時にAIが即チェック
    -   怪しいリンク/文言を検知 → 高齢者に通知せずブロック + 家族へ警報
3.  **家族登録・合言葉機能**
    -   家族がアプリ内で「声紋登録」「合言葉登録」（例：「今日はいい天気だね」）
    -   声クローン対策：生成AI音声検知（NTT東日本が開発中の技術を参考）

**Should（MVP後すぐ追加）**

-   本人確認後「かけ直しボタン」自動表示
-   通話録音 + AI要約（後で家族が確認可能）
-   緊急時「即本人接続」モード（家族からリモートON）

**Could（後回し）**

-   警察庁推奨アプリ連携（番号DB共有）
-   地域別詐欺トレンド学習（AIが自動アップデート）

#### **3. ユーザーストーリー（主要なもの）**

-   As a 高齢者, I want AIが代わりに電話に出てくれる so that 詐欺に引っかからない
-   As a 高齢者, I want 安全な家族の電話だけ通知される so that 大切な連絡を逃さない
-   As a 家族, I want 高齢者のスマホの着信履歴とAI判定結果をリアルタイムで見られる so that 安心できる
-   As a 高齢者, I want 簡単な設定（大きな文字・音声ガイド） so that 自分で操作できる

#### **4. 非機能要件**

-   **対応OS**：Android 10以上 / iOS 15以上（両方対応）
-   **プライバシー**：通話内容は端末内でAI処理（クラウド送信は最小限・暗号化）
-   **UI/UX**：超シンプル（大きな文字、1タップ操作、音声読み上げ）
-   **オフライン対応**：基本AIは端末内モデル（軽量LLM）で動作
-   **バッテリー**：常時監視でも1日持つ（省電力モード必須）

#### **5. 技術スタック候補（フェーズ2で確定）**

-   フロント：Flutter（1コードでiOS/Android）
-   通話制御：Android Telephony + iOS CallKit
-   AI音声応答：リアルタイムボイスAPI（2026年現在最強候補：Grok Voice / Claude Voice / Gemini Live）
-   声紋・詐欺判定：生成AI + 声クローン検知モデル
-   バックエンド：Supabase（無料・簡単） or Firebase
-   データベース：家族登録・合言葉・学習履歴

#### **6. 収益モデル（検討中）**

-   基本無料（警察庁推奨アプリ狙い）
-   プレミアム：無制限録音・家族複数人監視（月額380円）
-   自治体・保険会社提携（無料配布）
